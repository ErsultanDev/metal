Полки и подставки из нержавейки


  • Подставка для кухонного инвентаря

    Код товара: 2581

    Подставки для кухонного инвентаря предназначены для использования на предприятиях общественного питания, а также на продуктовых складах и магазинах, для временного складирования кухонного инвентаря и продуктов питания. Могут служить подставкой под котлы с первыми блюдами.

    Перейти


  • Подставка для пароконвектомата

    Код товара: 2582

    Подставка для пароконвектомата имеет направляющие, позволяющие разместить в ней 14 гастроемкостей типа GN-1/1 или 28 гастроемкостей GN-1/2. Также подставка имеет полку, на которую можно положить необходимые предметы.

    Перейти


  • Полка закрытая купе

    Код товара: 2583

    Полка закрытая кухонная купе (ПЗК) предназначена для длительного хранения продуктов, инвентаря и посуды в закрытом объеме.

    Перейти

    • Для расчета стоимости полок из нержавейки позвоните по телефону:
    8(727) 344-99-00; +7 (701) 266-77-00




  • Полка кухонная для досок / для крышек

    Код товара: 2584

    Полка кухонная для крышек ПКК предназначена для сушки и хранения крышек кастрюль и баков в моечных отделениях, горячих цехах.Полка для разделочных досок ПКД предназначена для хранения разделочных досок. Полка ПКД аналогична по конструкции с полкой для крышек, но имеет более широкие ячейки.

    Перейти


  • Полка настенная полуоткрытая

    Код товара: 2585

    Полка настенная полуоткрытая ПНП предназначена для открытого хранения и демонстрации продуктов, требующих постоянной вентиляции, а также наиболее часто используемой посуды и инвентаря.

    Перейти


  • Полка настенная для тарелок

    Код товара: 2586

    Полка состоит из кассеты для тарелок и поддона. Кассета представляет собой решетку из прутка, в ячейки которой помещают тарелки. В поддоне предусмотрен небольшой уклон, обеспечивающий сток жидкости, поступающей с мокрой посуды к сливному отверстию.

    Перейти

    • Для расчета стоимости полок из нержавейки позвоните по телефону:
    8(727) 344-99-00; +7 (701) 266-77-00




  • Полка настенная решетчатая

    Код товара: 2587

    Полка настенная решетчатая ПН-Р предназначена для хранения и временной расстановки посуды и кухонного инвентаря в предприятиях общественного питания, магазинах, заготовочных предприятиях.

    Перейти
