Изготовление мебели любой сложности

«MORRIS» - коллекция мебели с текстурой под экзотическое дерево «макассар»

Основной материал коллекции мебели для руководителя «MORRIS» - ЛДСП с глянцевым покрытием, имитирующим дерево макассар (эбонитовое дерево). Серию отличает солидная массивность, лаконичные очертания и продуманная эргономика изделий.

Серия Simple — лидер в сегменте недорогой мебели для персонала.

Эту серию отличают неординарность цветовых решений и высокая функциональность. Стильная современная коллекция Simple идеально подходит для формирования недорогого офисного пространства, а использование материалов и комплектующих, применяемых, как правило, в мебели более высокого ценового сегмента, гарантирует долгий срок ее службы.

Мебель для персонала Эдем

Мебель для персонала «ЭДЕМ» - прекрасная мебель для офиса, которая способна дополнить самый интересный и эксклюзивный интерьер. Благодаря плавным формам и линиям эта офисная мебель - идеальный вариант для оформления рабочего пространства творческих людей, а также для помещений, применяющихся для работы с посетителями и клиентами. Кроме этого, данная мебель прекрасно подходит для помещений и кабинетов небольшой площади, потому как она очень комфортная, эргономичная и функциональная. Так, например, офисный стол из набора «ЭДЕМ» требует немного пространства, но, вместе с этим, позволяет удобно и функционально расположить оргтехнику, оставив достаточно свободного пространства для документов и канцелярских принадлежностей. Удобные офисные шкафы из набора «ЭДЕМ» хорошо подойдут для хранения личных вещей сотрудников и офисных бумаг. Необходимо заметить, что мебель для офиса из комплекта «ЭДЕМ» может быть исполнена в трех цветах – это замечательный вариант для любых помещений и кабинетов.

Мебель для персонала Лайт

Если Вы цените индивидуальный стиль и непревзойденное качество продукции - мебель для офиса серии «Lite» идеально Вам подойдет. В нашей компании «Офисная мебель №1» Вы можете купить офисную мебель по приемлемым ценам. Эта серия подходит для оборудования офисов различной площади. Для создания уникального интерьера офиса Вы можете выбрать вариации мебели в разнообразных цветовых вариантах. Данная серия позволяет грамотно обустроить пространство и обеспечить комфортные условия для работы. Покупка офисной мебели «Lite» - это отличное решение!

Офисная мебель TREND

На первом месте должен быть комфорт - именно эта идея "красной линией" проходит через весь концепт мебельной коллекции Trend. Удобная и, что немаловажно, универсальная конструкция столов подходит для организации групповых и индивидуальных рабочих мест. Продуманная эргономика делает мебель Trend удобной для повседневного использования. Вся коллекция мебели Trend изготовлена из ламинированного ДСП высокого качества. Это не дорогой, легкий и неприхотливый материал, отличается высокой прочностью. В сумме все эти показатели делают мебель Trend отличным примером сочетания цены и качества.

Мебель для персонала «Гамма»

Оперативная мебель «Gamma» создана для динамичных компаний с активными сотрудниками. При создании коллекции учтены последние тенденции в мире офисной мебели — новейшие технологии, современные материалы и фурнитура, востребованные цветовые решения, эргономичные модели. Серия Gamma — отличная возможность создать в офисе приятную атмосферу и условия для эффективного труда ваших сотрудников.
