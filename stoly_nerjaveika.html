Столы из нержавейки


  • Стол, нержавеющая сталь, открытый

    Код товара: 2501

    Стол открытый предназначен для разделывания и обработки пищевых продуктов, а также для установки кухонного оборудования в предприятиях общественного питания, магазинах, заготовочных предприятиях.

    Перейти


  • Стол, нержавеющая сталь, закрытый
    Код товара: 2502

    Стол закрытый предназначен для использования в качестве профессионального стола, а также для хранения посуды и кухонного инвентаря в предприятиях общественного питания, магазинах, заготовочных предприятиях.

    Перейти


  • Стол с мойкой для обработки овощей
    Код товара: 2503

    Столы изготавливаются из нерж. стали, имеющей доступ к контакту с продуктами. Каркас сварной. Ножки регулируются по высоте. В столешницу по центру вварена ванна (сварная). В столешнице предусмотрено технологическое отверстие под смеситель. В комплект поставки входит гофросифон.

    Перейти

    • Для расчета стоимости столов из нержавейки позвоните по телефону:

    8(727) 344-99-00; +7 (701) 266-77-00



  • Стол для чистки рыбы
    Код товара: 2505

    Столы для чистки рыбы изготавливаются из нержавеющей стали, имеющей доступ к контакту с продуктами. Каркас сварной. Регулируемые по высоте ножки. На столешнице слева от ванны располагается отверстие для крепления душ-стойки (или смесителя). Душирующее устройство и гофросифон для ванны входят в комплект поставки.

    Перейти


  • Обвалочный стол
    Код товара: 2506

    Обвалочные столы изготавливаются из нержавеющей стали, имеющей доступ к контакту с продуктами. Каркас сварной. Регулируемые по высоте ножки. Для столов длиной от 1400 мм в средней части стола для усиления конструкции устанавливаются дополнительные ножки.

    Перейти


  • Стол разделочный, полка сплошная
    Код товара: 2507

    Столещница и полка стола разделочно-производственного изготовлены из нержавейки t=0,5 AISI 430, столешница усилена с внутренней стороны листом ламинированной древесностружечной плиты (ЛДСП), что увеличивает прочность и исключает прогиб столешницы.Каркас оцинкованный, стойки-уголок

    Перейти

    • Для расчета стоимости столов из нержавейки позвоните по телефону:

    8(727) 344-99-00; +7 (701) 266-77-00



  • Стол разделочный, полка-решетка
    Код товара: 2508

    Столы разделочно-производственные предназначены для разделывания и обработки пищевых продуктов, а также для установки кухонного оборудования в предприятиях общественного питания, магазинах, заготовочных предприятиях.

    Перейти


  • Стол-тумба купе
    Код товара: 2509

    Столы-тумбы купе (СТК) предназначены для использования в качестве профессионального стола и предназначены для разделки и последующей обработки пищевых продуктов. Благодаря наличию закрытого объема, столы-тумбы купе обеспечивают аккуратный внешний вид кухни и используются для хранения посуды, инвентаря, столовых приборов и сухих продуктов.

    Перейти

  • Стол-тумба купе с мойкой
    Код товара: 2510

    Все детали стола тумбы-купе с мойкой изготовлены из нержавеющей стали AISI 430. Детали разной толщины. Мойка изготовлена из из нержавеющей стали AISI 304. Гофросифон для слива воды и пробка для мойки в комплекте. Мойка без отверстия под смеситель.

    Перейти


  • Стол разделочно-производственный, стойки - труба
    Код товара: 2511

    Столешница и полка стола изготовлены из нержавейки t=0,5 AISI 430, столешница стола усилена ЛДСП t=16мм; стойки - труба d=40, t=1,0, каркас оцинкованный.

    Перейти
